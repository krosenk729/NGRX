#### Project

1. `git clone `
1. `cd <redux-app>` 
1. `yarn install` or `npm install`
1. Open src > App.js and replace arealwebsite.io with a _real_ endpoint
1. `yarn run` or `npm run`

Visit `localhost:3000` to start building.
